Welcome to jjmodel's documentation!¶

jjmodel (Just-Jahreiß model) is a Python package for a semi-analytic model of the Milky Way (MW) disk. The latest version is designed for modeling Galactocentric distances from approximately 4 kpc to 14 kpc and heights up to about 2 kpc. This code is complemented by three alternative sets of isochrones (PARSEC, MIST, and BaSTI), such that users can synthesize stellar populations and select observable samples on the colour-magnitude diagram.

Check out the content section for further information, including Installation for instructions on how to install the package. Also read section Acknowledgements for the list of publications and the way of acknowledging this code.

Note

This project is under active development.

If you have questions, suggestions, requests, or want to report a bug, please feel free to contact the developers (see Authors and contact).
